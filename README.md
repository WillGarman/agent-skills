# Agent Skills

Reusable skills for AI coding agents. Install with:

```
npx skills add WillGarman/agent-skills
```

## Skills

### [do-you-need-useEffect](./skills/do-you-need-useEffect/)

Strict useEffect discipline for React. Treats direct `useEffect` as banned until proven necessary.

**Sources:**
- [React: You Might Not Need an Effect](https://react.dev/learn/you-might-not-need-an-effect)
- [@ArcjetHQ: Why we banned React's useEffect](https://x.com/ArcjetHQ/status/1901668174789853267) (1.7M views, Mar 2026)
- Real production infinite-loop bugs from unstable callbacks in dep arrays

### [make-it-readable](./skills/make-it-readable/)

Code readability cleanup pass. Makes code skimmable before every PR.

**Source:**
- [@gabriel1: make code skimmable, run a separate cleanup prompt before every pr](https://x.com/gabriel1/status/1902049241707389100) (54K views, Mar 2026)
